# Vercel-Backend
# Nutri Assist backend (Vercel)

Endpoints:
- POST /api/saveGoogleFormSubmission

Environment variables required on Vercel:
- FIREBASE_SERVICE_ACCOUNT  → JSON contents of Firebase service account (string)
- FRONTEND_BASE_URL        → e.g. https://your-frontend.web.app (optional, used in links)

Deploy: import repo to Vercel or use `vercel` CLI.
